

NUCLEI CHEATSHEET A small addition for those who have read my article "Using Nuclei for OSINT.


NUCLEI CHEATSHEET
10 MAIN FLAGS
SCAN URL
$ NUCLEI -U EXAMPLE.COM
SCAN LIST OF URLS FROM TEXT FILE
$ NUCLEI -L URLS.TXT
SCAN USING TEMPLATE FROM .YAML FILE (OR DIR)
$ NUCLEI -U EXAMPLE.COM -T EMAIL.YAML
-TAGS
SCAN USING TEMPLATES WITH CERTAIN TAGS
$ NUCLEI -U EXAMPLE.COM -TAGS OSINT, TECH
-0
SAVE RESULTS IN FILE
$ NUCLEI -U EXAMPLE.COM -O RESULTS.TXT
-JSON
SAVE RESULTS IN JSON FORMAT
$ NUCLEI -U EXAMPLE.COM -0 RESULTS.TXT -JSON
-VAR
ASSIGN A VALUE TO A VARIABLE
$ NUCLEI -TAGS OSINT -VAR USER=IVANOV
-RL
RATE LIMIT PER SECOND (SPEED UP OR SLOW UP SCANNING, DEFAULT 150)
$ NUCLEI -U EXAMPLE.COM -RL
300
-S
SCAN USING TEMPLATES WITH CERTAIN SEVERITY: INFO, LOW, MEDIUM, HIGH, CRITICAL, UNKNOWN
$
NUCLEI -U EXAMPLE.COM -S
INFO
--HELP
GET A LIST OF ALL FLAGS
$ NUCLEI --HELP
